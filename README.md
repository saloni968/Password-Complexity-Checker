# Password Complexity Checker

## Description
A Python program that checks the strength of a password based on:
- Minimum length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

## Features
- Checks password complexity
- Displays password strength
- Easy to use
- Written in Python

## Requirements
- Python 3.x

## How to Run

1. Clone the repository:
   git clone https://github.com/saloni968/Password-Complexity-Checker.git

2. Navigate to the project folder:
   cd Password-Complexity-Checker

3. Run the program:
   python password_checker.py
